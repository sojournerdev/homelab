# tinycloud

tinycloud is my personal homelab K8s platform that I am running on a desktop that I repurposed. I think my mindset right now is to try and build a platform that feels cloud-native.
Planning to run K3s as the distro and use that to build on top a platform API, and would
be even cooler to extend the cluster to a small shop cloud provider.

This is meant to enjoy building something and continue learning in my own time. Let's see
how that goes! Maybe start a blog to track my progress?

## Goals

- Build a small K8s platform for a homelab.
- Use K3s as the K8s distribution.
- Use Ansible for host bootstrap and K3s installation.
- Use Flux for GitOps once the cluster is running.
- Build a platform API on top to create that platform engineering exp.
- Future future, think about extending the cluster to use a cloud provider for nodes.

## Current Status

This platform is in early development.
